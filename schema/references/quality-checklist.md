# 质量自检清单

## 一、三层结构完整性检查

### Part A — Self Memory
- [ ] 核心身份信息完整？（姓名、年龄、职业、城市、教育背景）
- [ ] 核心价值观有具体描述？（不是空洞的形容词）
- [ ] 生活习惯有具体描述？
- [ ] 重要记忆有时间线和影响？
- [ ] 人际关系图谱有具体场景？
- [ ] 成长轨迹有具体变化？

### Part B — Persona
- [ ] Layer 0 硬规则完整？（7条）
- [ ] Layer 1 身份信息完整？
- [ ] Layer 2 说话风格有具体特征？（口头禅、标点、emoji等）
- [ ] Layer 3 情感模式有具体描述？（开心、难过、生气、焦虑时的表现）
- [ ] Layer 3 决策模式有具体描述？
- [ ] Layer 4 人际行为有具体描述？

### Part C — Thinking Framework
- [ ] 心智模型有3-7个？
- [ ] 每个心智模型都有三重验证？（跨域复现、生成力、排他性）
- [ ] 每个心智模型都有证据？（至少2个场景）
- [ ] 每个心智模型都有局限？
- [ ] 决策启发式有5-10条？
- [ ] 每条决策启发式都有案例？
- [ ] 表达DNA有6个维度？（句式、词汇、节奏、幽默、确定性、引用）
- [ ] 价值观与反模式完整？

---

## 二、心智模型质量检查

### 三重验证
- [ ] 每个模型都在≥2个不同领域出现？（跨域复现）
- [ ] 每个模型都能推断对新问题的立场？（生成力）
- [ ] 每个模型都不是所有人都这样想？（排他性）

### 模型质量
- [ ] 模型数量在3-7个之间？
- [ ] 模型之间有张力但不矛盾？
- [ ] 模型有明确的应用场景？
- [ ] 模型有明确的局限性？

### 证据质量
- [ ] 每个模型都有至少2个不同场景的证据？
- [ ] 证据来自用户的真实表达？（不是编造的）
- [ ] 证据有具体场景描述？

---

## 三、决策启发式质量检查

- [ ] 每条规则都是「如果X，则Y」的格式？
- [ ] 每条规则都有具体案例支撑？
- [ ] 规则可以被新情况触发？（不只适用于原始案例）
- [ ] 规则数量在5-10条之间？
- [ ] 规则覆盖了主要决策场景？

---

## 四、表达DNA质量检查

### 辨识度
- [ ] 读起来有辨识度，不像通用AI？
- [ ] 没有过度模仿变成 caricature？
- [ ] 抓住了核心特征而非表面模仿？

### 完整性
- [ ] 句式偏好明确？（长句/短句、疑问/陈述）
- [ ] 词汇特征明确？（高频词、禁忌词）
- [ ] 节奏感明确？（先结论还是先铺垫）
- [ ] 幽默方式明确？（讽刺/自嘲/荒诞/冷幽默/不幽默）
- [ ] 确定性表达明确？（「我不确定」型 还是 「很明显」型）
- [ ] 引用习惯明确？

---

## 五、价值观与反模式检查

- [ ] 核心价值观有3-5条？
- [ ] 每条价值观都有具体描述？（不是空洞的形容词）
- [ ] 反模式有具体描述？
- [ ] 内在矛盾有至少2对？
- [ ] 矛盾是真实存在的？（不是编造的）

---

## 六、诚实边界检查

- [ ] 明确写了做不到什么？
- [ ] 标注了数据来源？
- [ ] 标注了分析时间？
- [ ] 承认了信息不足的维度？
- [ ] 至少有3条具体局限？

---

## 七、整体质量检查

### 可运行性
- [ ] 角色扮演规则清晰可执行？
- [ ] 回答工作流清晰可执行？
- [ ] 硬规则不会被低层覆盖？

### 真实性
- [ ] 用用户的眼睛看一个新问题，能得到有价值的视角？
- [ ] 不是用户原话的拼凑，而是框架的运行？
- [ ] 删掉名字后，还能认出这是谁的思维方式？

### 独特性
- [ ] 有用户的独特视角？（不是所有人都这样想）
- [ ] 有用户的独特表达？（不是通用AI味）
- [ ] 有用户的独特矛盾？（不是完美无缺）

---

## 八、不通过信号

### 心智模型
- ❌ 模型数量 <3 或 >10
- ❌ 模型没有三重验证
- ❌ 模型没有证据
- ❌ 模型没有局限

### 表达DNA
- ❌ 读起来像通用ChatGPT
- ❌ 过度模仿变成 caricature
- ❌ 只有表面模仿，没有核心特征

### 价值观
- ❌ 价值观都是空洞的形容词
- ❌ 没有内在矛盾（太假）
- ❌ 反模式不具体

### 诚实边界
- ❌ 只有「不能替代本人」
- ❌ 没有标注数据来源
- ❌ 没有承认信息不足的维度

---

## 九、自检流程

1. **结构检查**：检查三层结构是否完整
2. **心智模型检查**：检查三重验证和证据
3. **决策启发式检查**：检查规则格式和案例
4. **表达DNA检查**：检查辨识度和完整性
5. **价值观检查**：检查具体性和矛盾
6. **诚实边界检查**：检查局限和来源
7. **整体检查**：检查可运行性、真实性、独特性

---

## 十、不通过时的处理

1. **心智模型不足**：回到 Phase 3.3，补充更多场景的证据
2. **表达DNA不清晰**：回到 Phase 3.2，从聊天记录中提取更多特征
3. **价值观不具体**：回到 Phase 3.1，从笔记和经历中提取更多细节
4. **诚实边界不足**：补充更多局限和来源信息

**迭代上限**：最多循环2次。如果2轮后仍有不通过项，在诚实边界中标注薄弱维度，交付当前最优版本。
